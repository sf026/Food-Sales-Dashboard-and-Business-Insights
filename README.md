# Food-Sales-Dashboard-and-Business-Insights

This project provides an interactive **Power BI dashboard** for analyzing food sales data.  
The dashboard helps understand **sales performance, product demand, peak sale timings, and payment trends**.

Key Features
- Total Sales, Orders, Quantity & Average Transaction KPIs
- Sales comparison between **Fast Food vs Beverages**
- Top-selling items visualization
- Sales trend based on **time of day**
- Payment type comparison (Cash vs Online)

  Data Preparation
- Removed duplicates and missing values
- Converted date & numeric columns to correct formats
- Standardized item and category names
- Created calculated columns & DAX measures (Total Sales, Profit, Margin)

 Insights Gained
- Fast food category generated higher revenue than beverages.
- Sales peaked during **evening** and **night** hours.
- Certain items like Sandwich and Frankie drove the majority of revenue.
- Cash payments were more common, but online transactions also significant.
